Write a compelling property listing description for a [property type] located at [address or area]. 

Key details:
- Bedrooms: [number]
- Bathrooms: [number]
- Square footage: [size]
- Key features: [list 3-5 standout features like "updated kitchen," "mountain views," "pool"]
- Neighborhood highlights: [nearby amenities, schools, parks]

Write in an engaging, professional tone that appeals to [target buyer type, e.g., "young families," "luxury buyers," "first-time homebuyers"]. Focus on lifestyle benefits, not just features. Keep it between 150-200 words.
